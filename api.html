API information

The API is available at https://wallen.co.
  • /api/all - returns information for all softwares.
  • /api/name - returns information for the queried software.
  • /api/name/version - returns version for the queried software.
Use same spelling as on the site.

Example: curl https://wallen.co/api/Pi-hole/version
Return: 5.3.1
